### {command}CommandParameters

<ResponseField name="store" type="ConfigStore (String)" required>
  The name of the ConfigStore as it appears in stores section of the coralated .configu file or a type of the ConfigStore to use the default Store Configuration.
</ResponseField>

<ResponseField name="set" type="ConfigSet (String)" required>
  The path of the ConfigSet hierarchy. Use an empty string or "/" for the root set.
</ResponseField>

<ResponseField name="schema" type="ConfigSchema (String)" required>
  The name of the ConfigSchema as it appears in the schemas section of the coralated .configu file or a path to an acutal .cfgu.json|yaml|yml file.
</ResponseField>

<ResponseField name="configs" type="ConfigsDict (Object)">
  "key=value" pairs. Use an empty value to delete a Config.
</ResponseField>

<ResponseField name="pipe" type="EvalCommandReturn (object)">
  A result of a previous EvalCommand to pipe into this command.
</ResponseField>
